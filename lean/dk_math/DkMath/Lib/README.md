# DkMath.Lib README

## 1. Purpose

- `DkMath.Lib.*`
  - A place to organize public candidate APIs extracted from `DkMath.*`.
  - Create a "stable intermediate layer" to be reused from research lemmas.
  - Establish naming conventions, dependency directions, and entry files with public release in mind.
  - In principle, do not introduce research-specific local considerations here.
