## 因数分解の展開（y を除いた版）

```txt
(x+y)^0 - y^0         = 0
(x+y)^1 - y^1         =                      1x^1
(x+y)^2 - y^2         =                 1x^2 + 2x^1y^1
(x+y)^3 - y^3         =            1x^3 + 3x^2y^1 + 3x^1y^2
(x+y)^4 - y^4         =       1x^4 + 4x^3y^1 + 6x^2y^2 + 4x^1y^3
(x+y)^5 - y^5         = 1x^5 + 5x^4y^1 + 10x^3y^2 + 10x^2y^3 + 5x^1y^4
```


## 一般化

$$
(x+y)^n - y^n = \sum_{i=0}^{n-1} \binom{n}{i} x^{n-i} y^i
$$

$$
(x+y)^n - x^n = \sum_{i=0}^{n-1} \binom{n}{i} x^i y^{n-i}
$$

### 商の形

$$
\frac{(x+y)^n - y^n}{x} = \sum_{i=0}^{n-1} \binom{n}{i} x^{n-1-i} y^i
$$

$$
\frac{(x+y)^n - x^n}{y} = \sum_{i=0}^{n-1} \binom{n}{i} x^i y^{n-1-i}
$$

## 応用例

### 冪乗２項差

```txt
(x+y)^2 - y^2 - x^2   =                        2x^1y^1
(x+y)^3 - y^3 - x^3   =                   3x^2y^1 + 3x^1y^2
(x+y)^4 - y^4 - x^4   =              4x^3y^1 + 6x^2y^2 + 4x^1y^3
(x+y)^5 - y^5 - x^5   =        5x^4y^1 + 10x^3y^2 + 10x^2y^3 + 5x^1y^4
```

x/y どちらでも割れる構造

```txt
((x+y)^2 - y^2 - x^2) / x =                       2x^0y^1
((x+y)^3 - y^3 - x^3) / x =                  3x^1y^1 + 3x^0y^2
((x+y)^4 - y^4 - x^4) / x =              4x^2y^1 + 6x^1y^2 + 4x^0y^3
((x+y)^5 - y^5 - x^5) / x =        5x^3y^1 + 10x^2y^2 + 10x^1y^3 + 5x^0y^4
```

```txt
((x+y)^2 - y^2 - x^2) / y =                       2x^1y^0
((x+y)^3 - y^3 - x^3) / y =                  3x^2y^0 + 3x^1y^1
((x+y)^4 - y^4 - x^4) / y =              4x^3y^0 + 6x^2y^1 + 4x^1y^2
((x+y)^5 - y^5 - x^5) / y =        5x^4y^0 + 10x^3y^1 + 10x^2y^2 + 5x^1y^3
```

整理すると、

```txt
((x+y)^2 - y^2 - x^2) / x =                       2y^1
((x+y)^3 - y^3 - x^3) / x =                  3x^1y^1 + 3y^2
((x+y)^4 - y^4 - x^4) / x =              4x^2y^1 + 6x^1y^2 + 4y^3
((x+y)^5 - y^5 - x^5) / x =        5x^3y^1 + 10x^2y^2 + 10x^1y^3 + 5y^4
```

```txt
((x+y)^2 - y^2 - x^2) / y =                       2x^1
((x+y)^3 - y^3 - x^3) / y =                  3x^2 + 3x^1y^1
((x+y)^4 - y^4 - x^4) / y =              4x^3 + 6x^2y + 4x^1y^2
((x+y)^5 - y^5 - x^5) / y =        5x^4 + 10x^3y + 10x^2y^2 + 5x^1y^3
```
