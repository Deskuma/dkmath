### 1. 無次元一般宇宙式の正体：二項展開の「割り算つき」恒等式

お主の式

\[
u^d + xG_{d-1}(x,u) = (x+u)^d
\]

は、要するに

\[
(x+u)^d - u^d = x\,G_{d-1}(x,u)
\]

という形じゃな。ここで \(G_{d-1}(x,u)\) は（お主が Lean で閉じたやつ）実質的に

\[
G_{d-1}(x,u)=\sum_{k=0}^{d-1}\binom{d}{k+1}x^{k}u^{d-1-k}
\]

という「二項展開の交差項をまとめて \(x\) で割ったもの」になっておるはずじゃ。
つまり **\((x+u)^d\) の中身を “Gap = u^d” と “Body = xG” に分解しただけ**。
ゆえに、ここは常に成り立つ（恒等式）わけじゃ。

---

### 2. 似て見えるが決定的に違う点：FLT には “交差項” が居座る

FLT の式は

\[
x^n + y^n = z^n
\]

で、もし「宇宙式と同じノリ」で \(z = x+y\) と置きたくなるが、そこが罠じゃ。

実際、

\[
(x+y)^n - x^n - y^n = \sum_{k=1}^{n-1}\binom{n}{k}x^k y^{n-k}
\]

で、右辺は（\(x,y>0\) なら）**全部正**。
つまり

\[
(x+y)^n > x^n+y^n \quad (n>1)
\]

なので、**\(z=x+y\) は絶対に無理**。
ここで「形が近い」は、むしろ FLT の“逃げ道”がどこにあるかを教えてくれる：

\[
z^n = x^n+y^n < (x+y)^n \ \Rightarrow\ z < x+y
\]

かつ当然 \(z>\max(x,y)\)。
だから解があるなら

\[
\max(x,y) < z < x+y
\]

という狭い檻の中に押し込められる。これは地味に効く制約じゃ。

### 3. 「構造解析」で本当に勝てる入口：因数分解と \(p\)-進評価（LTE）

お主の \(G\) は「割り算つき差分」＝離散微分のようなものじゃが、FLT 側にも“差分構造”がある。

たとえば奇素数 \(p\) について

\[
x^p+y^p=(x+y)\bigl(x^{p-1}-x^{p-2}y+\cdots -xy^{p-2}+y^{p-1}\bigr)
\]

これは「交互符号の巨大な \(G\)」みたいなものじゃな（実は円分多項式の影がいる）。
ここで超強力な道具が **LTE（Lifting The Exponent）**：

- \(p\) が奇素数で、\(p\mid (x+y)\) かつ \(p\nmid xy\) なら

\[
v_p(x^p+y^p)=v_p(x+y)+v_p(p)
\]

つまり

\[
v_p(x^p+y^p)=v_p(x+y)+1
\]

（\(v_p(p)=1\) だから）。

もし FLT の反例があるとして

\[
x^p+y^p=z^p
\]

なら左辺の素因子指数は **“\(x+y\)” に強く縛られる**。
ここが「宇宙式の分解で不変量を縛る」発想と噛み合う可能性が高い。

### 2. 本丸：\(A^d-B^d\) は **だいたい \(n\) 乗にならない**

> つまり、その素数 \(p\) は \(a^k-b^k\)（\(k<n\)）を割らないのに \(a^n-b^n\) を割る。

これが何を意味するかというと――
**\(a^n-b^n\) には “指数 1 回だけ出る素因子” が混ざりやすい**、ということじゃ。

そして \(x^n\)（\(n>1\)）なら、素因子指数は全部 \(n\) の倍数（\(v_p(x^n)=n\,v_p(x)\)）でなければならん。
ゆえに、

- \(a^{md}-b^{md}\) に指数 1 の素因子が出る
- しかし \(x^n\) なら指数は必ず \(n\) の倍数

この時点で矛盾が出る可能性が極めて高い。

- \(A^d-B^d\) が \(n\) 乗であるには、**素因子指数の“全部”が \(n\) の倍数**である必要がある。
- だが \(A^d-B^d\) は差の形なので、ジグモンディ由来の「新規素因子」が**薄い指数（しばしば 1）**で刺さってくる。

## 命題（狙う否定筋）

> **定理（最短版）**
> \(d\ge 2\)、\(x>0\)、\(u>0\) を整数（自然数でも可）とすると
> \[
> (x+u)^d-u^d \ne x^d
> \]

> したがって
> \[
> xG_{d-1}(x,u) \ne x^d
> \]

> すなわち「二項定理項は \(x^d\) という形（同じ \(x\) を底にした純粋冪）に潰れない」。

## 証明（1行で決まるやつ）

二項定理より

\[
(x+u)^d
= x^d + \binom{d}{1}x^{d-1}u + \binom{d}{2}x^{d-2}u^2 + \cdots + u^d.
\]

両辺から \(u^d\) を引けば

\[
(x+u)^d-u^d
= x^d + \binom{d}{1}x^{d-1}u + \binom{d}{2}x^{d-2}u^2 + \cdots + \binom{d}{d-1}xu^{d-1}.
\]

ここで \(d\ge 2\) かつ \(x>0,u>0\) だから、右辺の追加項は全部 **正**。よって

\[
(x+u)^d-u^d > x^d.
\]

ゆえに等号は不可能、すなわち

\[
(x+u)^d-u^d \ne x^d.
\]

終わりじゃ。美しいほど一撃じゃろ。

## これを「\(G_{d-1}\) の言葉」に戻すと

お主の定義（宇宙式分解）により

\[
G_{d-1}(x,u)=\frac{(x+u)^d-u^d}{x}
\]

なので、上の不等式を \(x>0\) で割れば

\[
G_{d-1}(x,u) > x^{d-1}.
\]

つまり

\[
xG_{d-1}(x,u)=x^d
\quad\Longleftrightarrow\quad
G_{d-1}(x,u)=x^{d-1}
\]

という“潰し”は、\(u>0\) の時点で即死する。
（例外は \(u=0\) のときだけで、この場合は \((x+0)^d-0^d=x^d\) で確かに等号になる。だが FLT の世界では \(y=u\neq 0\) を想定するから、使えぬ。）

---

## 次の一手（この否定筋をFLT側へ繋ぐための形）

今の定理は「**同じ底 \(x\) の純粋冪 \(x^d\) には潰れない**」を確定した。
お主が次に狙うなら、筋が良いのはこの強化じゃ：

\[
(x+u)^d-u^d \text{ は } d\text{乗（一般の }t^d\text{）にもなりにくい}
\]

という方向（原始素因子・指数評価で叩く）へ進むと、FLT 的な「完全冪の不可能性」に直結しやすい。

だがまずは一本目の目的地、**「\(xG_{d-1}(x,u)\neq x^d\)」** は、今ので完全に閉じたぞい。
お主の宇宙式は、やはり“交差項が消えない”という真理を、きれいに露呈させる道具じゃな。

> 👩‍⚕️私D.:

おお！残るは、x^d を整数とした時に、(x+u)^d-u^d の整数解は難しい→出来ない。か。

> 🐺賢狼:

うむ、その「次の目的地」は綺麗に一本化できるぞ。
ただし、ここで賢狼が **真実の看板** を立てておく。

お主が言っている「出来ない」を“数学的に正確な形”に直すと、こうなる：

\[
(x+u)^d - u^d = t^d \quad (x,u,t\in\mathbb{Z}_{>0})
\]

これを
\(a:=x+u,\ b:=u\) と置き換えると、

\[
a^d - b^d = t^d
\quad\Longleftrightarrow\quad
a^d = b^d + t^d
\]

……お気づきかの？
**これは指数 \(d\) のフェルマー型方程式そのもの**じゃ。
つまり「この否定筋を一般 \(d>2\) で証明する」＝「FLT（少なくとも指数 \(d\) について）を証明する」に直結する。
目的地は正しいが、難所はまさにそこじゃよ。

---

## まず重要：\(d=2\) は「出来ない」どころか無限に出来る

\[
(x+u)^2-u^2 = x^2+2xu
\]

が平方になる例は山ほどある。たとえば \(x=1,u=4\) で

\[
(1+4)^2-4^2=25-16=9=3^2
\]

ゆえに「出来ない」は **\(d=2\) では偽**。
このケースは本質的に

\[
a^2-b^2=t^2
\]

＝ピタゴラス三つ組の世界じゃ。

---

## なので狙うべき定理はこうじゃ（\(d>2\) 専用）

> **目標定理（本命）**
> \(d>2\) とし、\(x,u,t\in\mathbb{Z}_{>0}\) に対して
> \[
> (x+u)^d-u^d=t^d
> \]

> は解を持たない。

これは「FLT の中でも、特に \(a-b=x>0\) という“差が正”の部分族を叩く」ことになる。
部分族だが、**原理的には FLT と同じ硬さ**を持ちうる（ただし攻め筋によっては、部分族だけ先に落とせる可能性はある）。

---

## ここから賢狼のおすすめ否定筋：原始素因子（Zsigmondy）＋指数（\(v_p\)）で潰す

お主の宇宙式の分解

\[
(x+u)^d-u^d = xG_{d-1}(x,u)
\]

は、「差」を扱うのに向いておる。
差の否定筋で王道はこれじゃ：

### ステップ1：原始素因子を呼ぶ

\(a>b>0\) のとき、多くの場合 \(a^d-b^d\) には「初めて現れる素因子」＝**原始素因子** \(p\) が存在する（Zsigmondy 型の主張）。

直感的には、

\[
a^d-b^d
\]

には、低い次数では割れない“新顔の素数”が混ざりやすい、ということじゃな。

### ステップ2：完全 \(d\) 乗なら指数は全部 \(d\) の倍数

もし

\[
a^d-b^d=t^d
\]

なら、任意の素数 \(p\) について

\[
v_p(a^d-b^d)=v_p(t^d)=d\,v_p(t)
\]

なので、左辺の素因子指数は **全部 \(d\) の倍数**でなければならぬ。

### ステップ3：原始素因子は“指数が薄い”ことを示して矛盾へ

原始素因子 \(p\) は普通、\(\Phi_d(a,b)\)（円分多項式の二変数版）側に現れる。
ここで狙う補題はこういう形になる：

- \(p\) が「原始素因子」で、さらに \(p\nmid d\) なら、典型的に
  \[
  v_p(a^d-b^d)=1
  \]

  あるいは少なくとも **\(d\) の倍数にはならない**。

そうすると完全 \(d\) 乗とは両立しない：

\[
v_p(a^d-b^d)=1 \not\equiv 0 \pmod d
\]

で即死じゃ。

> ここが“勝負の一点”。
> 「新しい素数が入ってくる」だけでは足りず、**その指数が \(d\) の倍数になれない**ことを詰める必要がある。

---

## お主の \(xG_{d-1}(x,u)\) でやるなら、命題の形はこれが美しい

お主が言う「\(x^d\) の整数組に書けるか？」を、より一般にして：

> **狙い命題（強い形）**
> \(d>2,\ x,u>0\) のとき
> \[
> xG_{d-1}(x,u)=(x+u)^d-u^d
> \]

> は **完全 \(d\) 乗 \(t^d\)** にならない。

この形なら、FLT に直結するし、宇宙式の分解の強み（差分の構造）も最大限に活きる。

---

## ただし注意点（賢狼の釘）

「原始素因子の指数が 1」を一般に言い切るには、円分多項式の性質や \(\gcd\) の評価（たとえば \(\gcd(\Phi_d(a,b),d)\) の制御）を要することが多い。
ここは Lean でも形式化しやすい部類の“補題の森”になるが、ちゃんと踏み固めれば一本道になる。

---

### まとめじゃ

- **\(d=2\)**：解は無限（出来ないは偽）。
- **\(d>2\)**：「\((x+u)^d-u^d=t^d\) は解なし」を示すのは、まさに **FLT 的核心**。
- お主の宇宙式は「差」を扱う道具として強い。
  **原始素因子＋指数（\(v_p\)）** で「完全冪になれない」を叩くのが、賢狼のおすすめの否定筋じゃ。

この路線で行くなら、次に作るべき“中核補題”は
**「原始素因子 \(p\) を取り、\(v_p((x+u)^d-u^d)\) が \(d\) の倍数にならぬ」**
これじゃな。ここが決まれば、城門は開くぞい。

> 👩‍⚕️私D.:

\(a^d-b^d\) のカタチって例の差の因数分解方程式ってやつだよね？

> 🐺賢狼:

うむ、まさにそれじゃ。お主の言う「差の因数分解方程式」そのものが

\[
a^d-b^d
\]

の定番分解じゃな。しかもこれが今回の否定筋（完全冪になりにくい）を叩く主戦場になる。

---

### 1. まず基本：差の冪の因数分解

任意の整数 \(a,b\) と正整数 \(d\) について

\[
a^d-b^d=(a-b)\bigl(a^{d-1}+a^{d-2}b+\cdots+ab^{d-2}+b^{d-1}\bigr).
\]

後ろの括弧を

\[
S_d(a,b):=\sum_{k=0}^{d-1}a^{d-1-k}b^k
\]

と書くと、綺麗に

\[
a^d-b^d=(a-b)\,S_d(a,b)
\]

じゃ。

---

### 2. お主の宇宙式との一致（ここが気持ちいい）

お主の状況では \(a=x+u,\ b=u\) だから

\[
a-b=x
\]

となって、

\[
(x+u)^d-u^d = x \, S_d(x+u,u).
\]

これが、まさにお主の

\[
(x+u)^d-u^d = xG_{d-1}(x,u)
\]

に対応する。つまり

\[
G_{d-1}(x,u)=S_d(x+u,u)
\]

（記号の取り方により同値）で、**「差の因数分解」＝「宇宙式の Body 分解」** になっておる。

---

### 3. もう一段：平方差・偶数差・円分分解（真打）

さらに \(d\) が合成数なら、差はもっと割れる：

- \(d=mn\) のとき

\[
a^{mn}-b^{mn}=(a^m-b^m)\Bigl(a^{m(n-1)}+a^{m(n-2)}b^m+\cdots+b^{m(n-1)}\Bigr).
\]

さらに深い構造としては「円分多項式」による分解がある：

\[
a^d-b^d=\prod_{m\mid d}\Phi_m(a,b)
\]

ここで \(\Phi_m(a,b)\) は二変数の円分多項式（\(\Phi_m(X,Y)\)）で、原始素因子（Zsigmondy）が出てくる場所がこの \(\Phi_d(a,b)\) 側じゃ。

---

### 4. なぜこの分解が否定筋に効くのか（今回の目的）

お主が叩きたいのは

\[
a^d-b^d=t^d
\]

の“不可能性”じゃな。すると

\[
(a-b)\,S_d(a,b)=t^d
\]

となる。

ここでよく使う基本観察は：

- \(\gcd(a-b,\,S_d(a,b))\) は「小さく抑えられる」ことが多い
  （典型的に \(d\) に関係する程度しか共有しない）
- だから積が \(d\) 乗になるには、各因子の素因子指数が“綺麗に \(d\) の倍数”に揃う必要がある
- ところが \(S_d\) 側（実は \(\Phi_d\) 側）に「新顔の素数」が薄い指数で混ざって破綻しがち

という流れになる。ここが、お主の狙う否定筋の骨格じゃ。

---

## まとめじゃ

- はい、\((a^d-b^d)\) はまさに **差の冪の因数分解**の定番。
- お主の宇宙式の \(xG_{d-1}(x,u)\) は、実質
  \[
  (a-b)\,S_d(a,b)
  \]

  の形で、差分因数分解と完全に一致する。
- そして否定筋は、この分解の上で
  **\(\gcd\) の制御**＋**原始素因子（\(\Phi_d\)）**＋**指数 \(v_p\)**
  を使って「\(d\) 乗になれぬ」を詰めるのが王道じゃ。

この先は、お主のスタイルに合わせて「次に立てる補題」を一本に切り出してやるのが良い。今の段階で一番キレが出るのは、

\[
\gcd(a-b, S_d(a,b)) \mid d
\]

みたいな補題（少し形は調整）を固めることじゃな。これが固まると、完全冪否定が一気にやりやすくなる。

> 🐺賢狼:

\[

\quad\text{（差の冪の因数分解）}
\]

これを Lean ではまず **差の冪の因数分解 lemma** として固める。そこが固まると、次に

- \(\gcd(a-b, S_d(a,b))\) の制御
- 「完全 \(d\) 乗になれぬ」へ（原始素因子など）

へ綺麗に繋がる。

### 1. 差の冪の因数分解：定義と lemma の器

```lean
import Mathlib

namespace DkMath

open scoped BigOperators
open Finset

/-!
`a^d - b^d = (a - b) * ∑ i in range d, a^(d-1-i) * b^i`
という「差の冪の因数分解」を “宇宙式の Body 項” として使う下地。

型はまず `CommRing`（減法が要る）で作り、必要なら `ℤ` に落とす。
-/

/-- 差の冪の因数分解に出る和：`∑_{i=0}^{d-1} a^(d-1-i) * b^i` -/
def diffPowSum {α : Type*} [CommRing α] (a b : α) (d : ℕ) : α :=
  ∑ i in range d, a^(d - 1 - i) * b^i

/--
差の冪の因数分解（目標）：

`a^d - b^d = (a - b) * diffPowSum a b d`

※ ここは mathlib 既存 lemma があるはず。
まずは `#check` で見つけて `simpa` で繋ぐのが最短じゃ。
-/
theorem pow_sub_pow_factor {α : Type*} [CommRing α] (a b : α) (d : ℕ) :
    a^d - b^d = (a - b) * diffPowSum a b d := by
  classical

  /-
  まず `#check` で既存 lemma を探す。

  例（環境により名前が違うことがある）：
  #check sub_eq_mul_geom_sum
  #check pow_sub_pow_eq
  #check sub_eq_mul_sum
  #check geom_sum_mul
  #check sub_eq_mul_geom_sum₂
  #check sub_eq_mul_geom_sum
  -/

  -- ★最優先：既存 lemma を見つけたら、ここを `simpa [diffPowSum, ...]` にする。
  -- 例（仮）：by simpa [diffPowSum, mul_comm, mul_left_comm, mul_assoc] using sub_eq_mul_geom_sum (a:=a) (b:=b) d

  -- ひとまず「器」だけ作っておく。見つけ次第差し替え。
  -- （ここに自前帰納証明を書くこともできるが、既存 lemma を使う方が堅牢。）
  sorry

/-- 宇宙式の Body 項：`(x+u)^d - u^d` を一般化したもの -/
def BodyPow {α : Type*} [CommRing α] (x u : α) (d : ℕ) : α :=
  (x + u)^d - u^d

/-- `BodyPow x u d` は差の冪として `(x) * (...)` に因数分解できる（a=x+u, b=u で a-b=x）。-/
theorem BodyPow_factor {α : Type*} [CommRing α] (x u : α) (d : ℕ) :
    BodyPow x u d = x * diffPowSum (x + u) u d := by
  -- (x+u)^d - u^d = ((x+u) - u) * diffPowSum (x+u) u d = x * ...
  simp [BodyPow, pow_sub_pow_factor, diffPowSum, sub_eq_add_neg, add_assoc, add_left_comm,
        add_comm, sub_eq_add_neg, mul_assoc, mul_left_comm, mul_comm]

end DkMath
```

この段階では、`pow_sub_pow_factor` の中身が `sorry` じゃが、ここを **mathlib 既存 lemma 一発**で閉じるのが最短じゃ。
お主の環境で `#check` を走らせれば、だいたい「それっぽい名前」が見つかる。見つかったら `simpa [diffPowSum] using ...` に差し替えて終い。

### 2. 次の目標（お主の「出来ない」へ）：いきなり一般否定は FLT 級、まずは局所否定を積む

お主が狙っているのは（本質的に）

\[
(x+u)^d-u^d = t^d \quad (d>2)
\]

の解なし。これはそのまま \(a^d=b^d+t^d\) で、**指数 \(d\) の FLT そのもの**に近い。

だから Lean では、次の順が強い：

1. **因数分解を確定**：上の `BodyPow_factor`
2. **\(\gcd\) 制御の補題**（ここが「完全冪を壊す」最初の関門）
   例：\(\gcd(a-b, S_d(a,b)) \mid d\) 型
3. その上で「もし \(t^d\) なら素因子指数が全部 \(d\) の倍数」をぶつける
4. （必要なら）原始素因子/Zsigmondy 系を入れて決着

Lean にすると、(2) を lemma として切り出すのが“次の一手”じゃ。

これは完全冪否定そのものではないが、探索空間を削る基本刃じゃ。

import Mathlib

namespace DkMath.Samples.SketchBook

open scoped BigOperators
open Finset

/-!
`a^d - b^d = (a - b) * ∑ i in range d, a^(d-1-i) * b^i`
という「差の冪の因数分解」を “宇宙式の Body 項” として使う下地。

型はまず `CommRing`（減法が要る）で作り、必要なら `ℤ` に落とす。
-/

/-- 差の冪の因数分解に出る和：`∑_{i=0}^{d-1} a^(d-1-i) * b^i` -/
def diffPowSum {α : Type*} [CommRing α] (a b : α) (d : ℕ) : α :=
  ∑ i ∈ range d, a^(d - 1 - i) * b^i

/--
差の冪の因数分解（目標）：

`a^d - b^d = (a - b) * diffPowSum a b d`

※ Mathlib を探ると `Finset.sum_range_choose` や幾何級数関連の lemma が見つかる。
自前証明で対応するか、既存 lemma を活用するか決定する。
-/
theorem pow_sub_pow_factor {α : Type*} [CommRing α] (a b : α) (d : ℕ) :
    a^d - b^d = (a - b) * diffPowSum a b d := by
  induction d with
  | zero =>
    simp [diffPowSum, range_zero, Finset.sum_empty]
  | succ d ih =>
    have eq1 : a^(d + 1) - b^(d + 1) = a * a^d - b * b^d := by ring
    rw [eq1]
    have eq2 : a * a^d - b * b^d = (a - b) * a^d + b * (a^d - b^d) := by ring
    rw [eq2, ih]
    have : diffPowSum a b (d + 1) = a^d + b * diffPowSum a b d := by
      unfold diffPowSum
      -- split the 0-th term and shift the rest (use sum_range_succ' to get f 0 + ∑ f (i+1))
      rw [Finset.sum_range_succ']
      -- show the tail sum equals b * the shifted sum
      have tail_eq : ∑ k ∈ range d, a ^ (d + 1 - 1 - (k + 1)) * b ^ (k + 1) =
                     b * ∑ i ∈ range d, a ^ (d - 1 - i) * b ^ i := by
        -- move the `b` inside the sum so `sum_congr` can match summands
        rw [Finset.mul_sum]
        apply Finset.sum_congr rfl
        intro i hi
        simp only [add_tsub_cancel_right, pow_succ, Nat.sub_sub, Nat.add_comm]; ring
      rw [tail_eq]
      grind
    rw [this]
    grind

/-- 宇宙式の Body 項：`(x+u)^d - u^d` を一般化したもの -/
def BodyPow {α : Type*} [CommRing α] (x u : α) (d : ℕ) : α :=
  (x + u)^d - u^d

/-- `BodyPow x u d` は差の冪として因数分解できる（a=x+u, b=u で a-b=x）。 -/
theorem BodyPow_factor {α : Type*} [CommRing α] (x u : α) (d : ℕ) :
    BodyPow x u d = x * diffPowSum (x + u) u d := by
  induction d with
  | zero =>
    simp [BodyPow, diffPowSum, range_zero, Finset.sum_empty]
  | succ d ih =>
    show BodyPow x u (d + 1) = x * diffPowSum (x + u) u (d + 1)
    simp only [BodyPow]
    have key : (x + u)^d - u^d = x * diffPowSum (x + u) u d := by
      have h := ih
      simp only [BodyPow] at h
      exact h
    have h : (x + u)^(d + 1) - u^(d + 1) =
        x * ((x + u) * diffPowSum (x + u) u d + u^d) := by
      have h1 : (x + u)^(d + 1) - u^(d + 1) =
          (x + u) * (x + u)^d - u * u^d := by ring
      have h2 : (x + u) * (x + u)^d - u * u^d =
          (x + u) * ((x + u)^d - u^d) + (x + u) * u^d - u * u^d := by ring
      have h3 : (x + u) * ((x + u)^d - u^d) + (x + u) * u^d - u * u^d =
          (x + u) * (x * diffPowSum (x + u) u d) + x * u^d := by
        rw [key]; ring
      have h4 : (x + u) * (x * diffPowSum (x + u) u d) + x * u^d =
          x * ((x + u) * diffPowSum (x + u) u d + u^d) := by ring
      rw [h1, h2, h3, h4]
    rw [h]

   p^k\mid \gcd(a-b,S_d)\Rightarrow p^k\mid d

   \gcd(a-b,S_d)\mid d

証明：

$x\cdot e^k = x\cdot e^{\log_e(x)} = x \cdot x = x^2$
$x\cdot e^k = e^{k \log_e(x)}$

つまり、指数対数の関係式を使うと、
$x\cdot G_{d-1}(x,u) = t^d$ の場合に
$\ln(x) + \ln(G_{d-1}(x,u)) = d \ln(t)$
が成り立つ。

だが、$G_{d-1}(x,u)$ は $x^{d-1} + ... + u^{d-1}$ であり、$t^d/x$ が整数になるとは限らない。
$x\cdot G_{d-1}(x,u)$ が完全な $d$ 乗になることはない。を、証明することができれば、FLT の一部を示すことができる。
