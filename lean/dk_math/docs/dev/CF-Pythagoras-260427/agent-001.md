### 主要な定義

- **`IsPythagoreanTriple a b c`**: ピタゴラス三つ組の述語 `a² + b² = c²`
- **`PythagoreanDifference₁/₂`**: 差分構造 `c² - a²`, `c² - b²`  
- **`PythagoreanCosmicForm a u`**: 宇宙式的表現 `2au + u²`

### 主要な定理

1. **`pythagoras_as_difference`**: 古典的ピタゴラスと差分形式の等価性

   ```lean
   a² + b² = c² ↔ c² - a² = b² ∧ c² - b² = a²
   ```

2. **`pythagoras_cosmic_form`**: `c = a + u` のとき

   ```lean
   (a + u)² - a² = 2au + u²
   ```

3. **`pythagoras_gap_beam_interpretation`**: Gap/Beam 解釈

   ```lean
   b² = u × (2a + u)  -- u が「Gap」、2a+u が「Beam」
   ```

4. **`parametrization_embeds_cosmic_structure`**: 古典的パラメトリゼーション `(m²-n², 2mn, m²+n²)` が差分構造を内包することの証明

### 重要な洞察

お主の研究ノートで議論された以下の解釈を厳密に証明できたのじゃ:

> ピタゴラスの定理とは、三辺の長さの関係ではなく、  
> 三辺の平方が互いに **差分生成関係** を成すという二次整合条件である。  
> 各短辺の平方は、斜辺平方から他方の短辺平方を引いた  
> 「**あるべき乗差**」として定義される。

特に、`c = a + u` と置くと `b² = 2au + u²` となり、これは宇宙式 `(x+u)² - x(x+2u) = u²` と構造的に関連することを示した。ただし、前者は「Beam 構造」で `2au + u²`、後者は恒等的に `u²` へ落ちるという違いがあることも明確化したぞ。
