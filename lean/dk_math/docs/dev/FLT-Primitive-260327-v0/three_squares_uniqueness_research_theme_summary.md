# 三平方構造によるユニーク性生成の研究テーマ整理

## 1. 研究主題

本テーマの中心は、従来の **Body（差分）中心の観測** から一歩進み、

\[
\text{Big},\quad \text{Gap},\quad \text{Core}
\]

という **三つの完成平方の相互関係** から、素数的ユニーク性・原始素因子・非整合構造を読み解くことである。

特に重要な転換は、

\[
\text{Body} = \text{Big} - \text{Gap}
\]

を単独対象として見るのではなく、

\[
\text{Body} = \text{三平方の相互非整合が露出した境界層}
\]

として再解釈する点にある。

---

## 2. 基本定義

二次の場合、基本構造を以下で固定する。

\[
\text{Big} := (x+u)^2
\]

\[
\text{Core} := x^2
\]

\[
\text{Gap} := u^2
\]

差分として

\[
\text{Body} := \text{Big} - \text{Gap} = x(x+2u)
\]

を定義する。

さらに Beam を

\[
\text{Beam} := 2ux
\]

と見れば、

\[
\text{Big} = \text{Core} + \text{Beam} + \text{Gap}
\]

という三層分解が得られる。

---

## 3. 問いの再定式化

従来の問いは

\[
\text{Body はなぜ平方にならないのか}
\]

であった。

本研究ではこれを、より構造的に

\[
\text{三つの平方は、どのように素因子的ユニーク性を生むのか}
\]

へ再定式化する。

すなわち対象は差そのものではなく、

- Big：総体平方
- Core：中心平方
- Gap：外部完成平方

の三者である。

---

## 4. ユニーク性生成の仮説

### 4.1. 個体ユニーク性

各平方は単独では

\[
v_p(\square) \equiv 0 \pmod 2
\]

を満たし、完全に整った偶数冪構造を持つ。

したがって、個体ごとに

- Big 固有の素因子集合
- Core 固有の素因子集合
- Gap 固有の素因子集合

が存在する。

### 4.2. 相互非整合

三平方の差を取ると、この偶数冪整合が崩れる。

例えば

\[
\text{Big} - \text{Gap} = x(x+2u)
\]

\[
\text{Big} - \text{Core} = 2ux + u^2
\]

ここで現れる素因子は、各平方の単独構造には直接見えず、

\[
\text{平方どうしの非整合によって初めて露出する素数}
\]

と解釈できる。

### 4.3. 原始素因子との接続

この露出素数は、高次では

\[
(x+u)^d - x^d,
\quad
(x+u)^d - u^d
\]

に現れる原始素因子と対応しうる。

したがって本テーマは、Zsigmondy 型の「新素数出現原理」を幾何分解で読み直す視点を与える。

---

## 5. Big を集合とみなす視点

Big は単なる一数ではなく、

\[
\text{Big} = \text{Core} + \text{Beam} + \text{Gap}
\]

として、全構成要素を含む **母集合的総体** とみなす。

このとき研究対象は

\[
\text{Big 内のどこで新しい素因子が露出するか}
\]

という配置問題になる。

特に、

- Core にのみ属する素数
- Gap にのみ属する素数
- Big 全体でのみ観測される素数
- 差分層 Body / Beam で初めて観測される素数

の分類が中心課題となる。

---

## 6. 研究の主要命題候補

### 命題 A. 三平方ユニーク性命題

\[
\text{Big},\text{Gap},\text{Core}
\]

の三平方が互いに異なるとき、差分層に少なくとも一つの新素因子が露出する。

### 命題 B. 差分境界素因子命題

\[
\text{Body} = \text{Big} - \text{Gap}
\]

に現れる素因子のうち、Big と Gap のどちらにも平方冪として整合しないものを「境界素因子」と定義する。

### 命題 C. 高次一般化

\[
\text{Big}_d := (x+u)^d,
\quad
\text{Core}_d := x^d,
\quad
\text{Gap}_d := u^d
\]

に対して、

\[
\text{Big}_d - \text{Gap}_d = x\,GN(d,x,u)
\]

の primitive prime は三完成体の非整合から必然的に生じる。

---

## 7. 今後の具体タスク

1. 二次での素因子比較表作成
   \[
   v_p(\text{Big}),\ v_p(\text{Gap}),\ v_p(\text{Core})
   \]
   を素数ごとに比較する。

2. Body / Beam で初出する素因子の定義
   「境界素因子」の厳密定義を置く。

3. 高次 \(d\) への一般化
   \[
   (x+u)^d,\ x^d,\ u^d
   \]
   の三完成体構造へ拡張する。

4. FLT / primitive packet への接続
   primitive prime selection を「三完成体の非整合から出る prime」として再定式化する。

---

## 8. 研究上の意義

このテーマは、従来の

\[
\text{差の因数分解}
\]

中心の数論から、

\[
\text{完成体の配置と非整合}
\]

中心の数論へ視点を反転する。

これにより、

- 原始素因子
- 差の冪
- GN / Beam
- packet descent
- 三平方の配置幾何

を統一的に扱える可能性がある。

とくに「新素数は差から出る」のではなく、

\[
\text{完成平方どうしのユニーク性衝突から露出する}
\]

という見方を与える点が、本研究の核である。
