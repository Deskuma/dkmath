## 今 Lean で証明しているもの

いま証明しているのは、

```text id="m9w2af"
q2 を保存する作用がある

```

ここまでじゃ。

これを幾何的に見れば「四分回転」っぽい。
複素数的に見れば「虚数単位」っぽい。
三角関数的に見れば「90°」っぽい。

だが、それらはすべて **後読み** じゃ。

## 円として見るには追加構造が要る

$$
q2(x,y)=x^2+y^2
$$

この式だけを見ると、代数的には「二成分の二次量」じゃ。
これを「円の半径二乗」と読むには、さらに、

```text id="a3jz3c"
x 軸と y 軸を独立な方向として読む
それらを直交方向として読む
同じ尺度で測る
Euclidean plane として解釈する
```

が必要になる。

つまり、Lean 内の数式はまだ、

```text id="6d7bgi"
q2 の level set
```

であって、即座に「円」とは限らない。

「円」と呼ぶのは、標準 Euclidean 解釈を載せた後じゃな。

## 90° はさらに後

お主の言う通り、\(90^\circ\) はもっと後じゃ。

まず「一周」を定義し、さらにそれを \(360^\circ\) と測る単位系を入れなければならない。

だから、いまの構造から直接出ているのは、

```text id="hur5kg"
90°
```

ではなく、

```text id="kpno43"
一周を 4 等分する作用
```

でもまだ少し幾何寄りじゃ。
さらに厳密には、

```text id="zambww"
exact order 4 の作用
```

じゃ。

これが最も安全。

## 言い換えるなら

```text id="kz3t7k"

```

この順番が大事。

## 文書ではこう書くのが安全

読者向けには、こう整理すると強い。

```text id="dddxo4"
The primary statement is not that the action is a 90-degree rotation.
The primary statement is that it is an exact-order-four q2-preserving action.
The geometric reading as a quarter-turn appears only after choosing the usual Euclidean interpretation of q2.
```

日本語なら、

```text id="acqt32"
主張の本体は「90 度回転」ではない。
主張の本体は「q2 を保存する exact order 4 の作用」である。
それを標準ユークリッド平面として読むと、後から 90 度回転に見える。
```

## ここが強い理由

これにより、DkMath は「円」や「角度」を仮定していない、と言える。

```text id="myd8nw"
円を前提にしない
角度を前提にしない
三角関数を前提にしない
複素数を前提にしない
```

それでも、

```text id="lekp2c"
4 回で戻る保存作用
```

が出る。

つまり、円や角度の説明に使われる構造の一部が、より原始的な代数層に存在している。

これは本当に重要じゃ。
