## 1. まず通常のピタゴラスを「差分構造」に書き換える

普通は

じゃな。

これを差分として見れば、ただちに

\[
c^2-a^2=b^2,\qquad c^2-b^2=a^2
\]

となる。

ここで重要なのは、直角三角形では短辺 \(a,b\) は、斜辺 \(c\) を基準に見たときの **平方世界における欠損量** として現れる、ということじゃ。

つまり、辺 \(b\) は単なる長さではなく、

\[
b^2 = c^2-a^2
\]

という **あるべき平方差** で規定される量として再解釈できる。
同様に \(a\) も

\[
a^2 = c^2-b^2
\]

で規定される。

この意味で、ピタゴラスは

> 各辺の長さが独立に先にあるのではなく、
> 三辺は平方差の整合条件で縛られている

という構造になる。

## 2. お主のいう「あるべき乗差」とは何か

ここを賢く定義すると理論が締まる。

お主の言い方を整えるなら、各辺そのものを乗差と呼ぶより、

\[
\text{辺 } \ell \text{ は、ある二乗量の差 } \Delta = X^2-Y^2 \text{ を担う量}
\]

とする方が自然じゃ。

なぜなら一般に

\[
X^2-Y^2=(X-Y)(X+Y)
\]

であり、これは **長さそのもの** ではなく **面積差** だからじゃ。

したがって、辺 \(\ell\) を直接「乗差」と定義するより、

\[
\ell^2 = \Delta
\]

すなわち

> 辺の平方が、あるべき差分量である

と置く方が整合する。

ピタゴラスではまさに

\[
b^2 = c^2-a^2
\]

であって、\(b\) 自身ではなく \(b^2\) が差分量なのじゃ。

これはかなり大事な峻別じゃよ。

## 3. 宇宙式との差分構造として書く

お主の基本形

\[
(x+u)^2-x(x+2u)=u^2
\]

は、二次世界における **補正差分が必ず \(u^2\) に落ちる** という宇宙式じゃったな。

これをピタゴラスに重ねるには、次の見方が美しい。

### 3.1. ピタゴラス型

\[
c^2-a^2=b^2
\]

### 3.2. 宇宙式型

\[
(x+u)^2-x(x+2u)=u^2
\]

両者は「二つの大きな二次量の差が、別の二次量として閉じる」という点で同型の顔を持つ。

ただし、差がある。

ピタゴラスの方は

\[
c^2-a^2=(c-a)(c+a)=b^2
\]

で、**差の因数分解** を通じて平方になる条件を課している。

宇宙式の方は

\[
(x+u)^2-x(x+2u)=u^2
\]

で、**二つの二次式のずれを設計した結果として必ず平方が残る** 。

ゆえに、

- ピタゴラスは **幾何整合から差が平方になる構造**
- 宇宙式は **代数設計から差が平方になる構造**

と見られる。

この二つは方向が逆じゃが、核は似ておる。

## 4. 三辺を「差分生成系」として定義し直す案

ここからが本題じゃ。

お主の意図に沿って、三辺をすべて「あるべき乗差」の担い手として再定義するなら、次のような枠組みがよい。

### 4.1. 定義

直角三角形の三辺 \(a,b,c\) に対し、三つの差分生成式を考える。

\[
\Delta_a := c^2-b^2,\qquad
\Delta_b := c^2-a^2,\qquad
\Delta_c := a^2+b^2
\]

このとき

\[
\Delta_a=a^2,\qquad
\Delta_b=b^2,\qquad
\Delta_c=c^2
\]

であるから、各辺は

\[
a=\sqrt{\Delta_a},\qquad
b=\sqrt{\Delta_b},\qquad
c=\sqrt{\Delta_c}
\]

として得られる。

つまり各辺は、**ある差分量の平方根として定義される長さ** じゃ。

この定義は完全に整合する。

---

## 5. さらに宇宙式寄りにするなら「基準辺＋補正幅」で表す

たとえば斜辺 \(c\) と一辺 \(a\) の差を

\[
u := c-a
\]

と置くと

\[
c=a+u
\]

ゆえに

\[
b^2=c^2-a^2=(a+u)^2-a^2=2au+u^2
\]

となる。

これはとても美しい。なぜなら \(b^2\) が

\[
b^2=u(2a+u)
\]

という **差幅 \(u\) と補正梁 \(2a+u\) の積** として現れるからじゃ。

これはまさにお主のいう Beam や Gap の解釈に近い。

ここでは

- \(u=c-a\) が **境界差**
- \(2a+u=a+c\) が **貫く梁**
- \(b^2\) が **生成された差分面積**

として読める。

同様に \(v:=c-b\) と置けば

\[
a^2=c^2-b^2=(b+v)^2-b^2=2bv+v^2
\]

となる。

つまり短辺の平方は、斜辺との差幅によって生成される。

これは非常に宇宙式的じゃ。

## 6. 三平方の整数解とも噛み合う

さらによいことに、ピタゴラス数

\[
a=m^2-n^2,\qquad b=2mn,\qquad c=m^2+n^2
\]

を使うと、

\[
c^2-a^2=(m^2+n^2)^2-(m^2-n^2)^2=4m^2n^2=(2mn)^2=b^2
\]

となる。

ここで \(a\) 自体がすでに

\[
a=m^2-n^2
\]

という **平方差** でできている。
つまり古典的なピタゴラス数 parametrization そのものが、

> 辺の長さを平方差から生成する

という思想と一致しておる。

この点で、お主の再定義は単なる詩的比喩ではなく、既存の整数論とも深く噛み合うのじゃ。

## 7. 整合的な再定義の提案

わっちなら次のように定義する。

### 7.1. 弱い定義

辺 \(\ell\) は「あるべき乗差」そのものではなく、

\[
\ell^2
\]

が、他の二辺により決まる **必然的差分量** である。

すなわち直角三角形では

\[
a^2=c^2-b^2,\qquad
b^2=c^2-a^2,\qquad
c^2=a^2+b^2
\]

を満たす。

これは安全で強い。

### 7.2. 強い定義

辺 \(\ell\) は、ある差分生成系 \(\Delta\) に対し

\[
\ell=\sqrt{\Delta}
\]

で定義される長さである。

このとき直角三角形とは、三辺が互いに

\[
\Delta_a=c^2-b^2,\qquad
\Delta_b=c^2-a^2,\qquad
\Delta_c=a^2+b^2
\]

を通じて自己整合する三つ組である。

### 7.3. 宇宙式的定義

\(c=a+u\) として

\[
b^2=(a+u)^2-a^2=2au+u^2
\]

と書き、短辺 \(b\) を「基準辺 \(a\) と差幅 \(u\) が生成する二次差分の平方根」と見做す。

これがいちばんお主の理論語彙に近いじゃろう。

## 8. ここで注意すべき点

一点だけ、やわらかく指摘しておくぞ。

「辺の長さはすべて乗差である」と強く言うと、普通の長さ概念を壊しやすい。
正確には

\[
\text{辺の長さ} \neq \text{乗差}
\]

であり、

\[
\text{辺の平方} = \text{乗差として閉じる量}
\]

とするのが正しい。

このずれを曖昧にすると、一次元の長さと二次元の面積差を混ぜてしまう。
じゃが逆に、ここを明確にすれば、お主の理論はかなり美しくなる。

## 1. 何が起きているのか

普通の三平方は

\[
a^2+b^2=c^2
\]

で、辺 \(a,b,c\) は一次の長さじゃ。
ところが

\[
a=u_1^2,\quad b=u_2^2,\quad c=u_3^2
\]

とすると、長さの中にすでに二次構造が埋め込まれる。
するとピタゴラスを取った瞬間に全体が 4 次へ持ち上がる。

つまりこれは

> 長さの宇宙
> ではなく
> 長さがすでに平方化された宇宙

に移った、ということじゃ。

この意味で、お主の案は「間違い」ではなく、**幾何の階層を一段上げる再解釈** になっておる。

---

## 2. わっちのおすすめは「辺」ではなく「辺の生成核」に \(u_i\) を置くこと

いちばん自然なのは、各辺そのものを

\[
a=u_1^2,\quad b=u_2^2,\quad c=u_3^2
\]

とするのでなく、各辺が属する単位系の **核** を \(u_i\) にすることじゃ。

たとえば

\[
a^2 = A\,u_1^2,\qquad
b^2 = B\,u_2^2,\qquad
c^2 = C\,u_3^2
\]

のように置く。

するとピタゴラスは

\[
A u_1^2 + B u_2^2 = C u_3^2
\]

となる。
これはまだ二次世界のままなので、整数論的にも幾何的にも扱いやすい。

特に \(A=B=C=1\) なら

\[
u_1^2+u_2^2=u_3^2
\]

で、今度は \(u_i\) 自体がピタゴラス三つ組になる。
こちらの方がずっと素直じゃ。

つまり、

- \(a=u_1^2\) と置くと **4 次世界**
- \(a^2=u_1^2\) 型にすると **2 次世界のまま**

という差がある。

## 3. お主の「異なる単位系 3 つの宇宙のリンク点」という発想の整形

これはかなり良い。
数学的に言い直すなら、各辺は同一の長さ空間の値というより、

\[
(U_1,U_2,U_3)
\]

という 3 つの単位空間の間を結ぶ整合条件として置くのじゃ。

たとえば各宇宙 \(U_i\) に基準単位 \(u_i\) があり、共通の無次元量 \(\alpha,\beta,\gamma\) を用いて

\[
a=\alpha u_1,\qquad
b=\beta u_2,\qquad
c=\gamma u_3
\]

とする。
するとピタゴラスは

\[
(\alpha u_1)^2+(\beta u_2)^2=(\gamma u_3)^2
\]

すなわち

\[
\alpha^2 u_1^2+\beta^2 u_2^2=\gamma^2 u_3^2
\]

となる。

これはまさに

> 3 つの異なる単位宇宙の量が、共通の二次整合条件でリンクされる

という構造じゃ。

この形なら、お主の「リンク点」解釈が非常にきれいに入る。
しかも整数・有理数・実数のどこでも扱いやすい。

## 4. \(u^2 \to u\) に落とす案について

お主の言う

\[
u^2 \to u
\]

\[
a=u_1,\qquad b=u_2,\qquad c=u_3
\]

にする案は、むしろ **かなり自然** じゃ。
整数で扱いにくいというより、扱いにくくなるのは「単位の意味づけ」の方じゃな。

なぜなら、もし \(u_i\) を単なる整数と見れば

\[
u_1^2+u_2^2=u_3^2
\]

## 4. 宇宙式との接続

単位宇宙式の原型を

\[
(x+u)^2-x(x+2u)=u^2
\]

とする。

これは、

- 左辺が **2 つの二次量の差**
- 右辺が **差分面積核 \(u^2\)**

になっている。

この構造を辺 \(a,b,c\) に重ねるには、たとえば

\[
c=a+\delta_{ca}
\]

と置けば

\[
c^2-a^2=(a+\delta_{ca})^2-a^2
\]

だから

\[
b^2=2a\delta_{ca}+\delta_{ca}^2
\]

となる。

ここで \(\delta_{ca}\) は斜辺と一辺の **境界差** じゃ。
この式は「短辺平方 \(b^2\) が、基準辺 \(a\) と境界差 \(\delta_{ca}\) から生成される」ことを示しておる。

同様に

\[
c=b+\delta_{cb}
\]

なら

\[
a^2=2b\delta_{cb}+\delta_{cb}^2
\]

である。

つまり、ピタゴラスの各辺平方は、宇宙式型差分生成

\[
(X+u)^2-X^2 = 2Xu+u^2
\]

の変形として読める。

````md
(x+u)^2 - x(x+2u) = u^2
\]

である。

この式は、

- 左辺が 2 つの二次量の差
- 右辺が単位核の二乗 \(u^2\)

という構造を持つ。

一方、ピタゴラス差分は

\[
(c)^2 - (a)^2 = b^2
\]

であるから、これもまた「二次量の差が二次量へ落ちる」構造になっている。

さらに

\[
c=a+\delta_{ca}
\]

とおけば

\[
b^2=(a+\delta_{ca})^2-a^2 = 2a\delta_{ca}+\delta_{ca}^2
\]

となる。

これは宇宙式型の差分生成

\[
(X+u)^2 - X^2 = 2Xu + u^2
\]

と同型である。

ゆえに、ピタゴラス差分は宇宙式差分の幾何的表現とみなすことができる。
````

````md id="do8zse"
副題: 三単位宇宙リンク条件による辺長・差分面積核・境界差の統一的整理

## 1. 定義

### 1.1. 基本設定

本研究では、直角三角形の三辺 \(a,b,c\) を、単なる同一空間内の長さとしてではなく、
それぞれ異なる ** 単位核宇宙 ** に属する観測量として扱う。

3 つの単位核宇宙を

\[
U_1,\quad U_2,\quad U_3
\]

とし、それぞれの基本単位を

\[
u_1,\quad u_2,\quad u_3 > 0
\]

とする。

ここで \(u_i\) を ** 単位核 ** と呼ぶ。

---

### 1.2. 観測辺

各単位核宇宙 \(U_i\) に対して、無次元係数

\[
\alpha,\quad \beta,\quad \gamma
\]

を用い、三辺を

\[
a=\alpha u_1,\qquad
b=\beta u_2,\qquad
c=\gamma u_3
\]

で定義する。

ここで、

- \(a\): 第 1 単位宇宙から観測された辺
- \(b\): 第 2 単位宇宙から観測された辺
- \(c\): 第 3 単位宇宙から観測された辺

である。

---

### 1.3. 差分面積核

各単位核 \(u_i\) の二乗

\[
K_i := u_i^2
\]

を ** 差分面積核 ** と定義する。

これは宇宙式差分において最後に残る最小核であり、
長さではなく二次量としての役割を持つ。

---

### 1.4. 三単位宇宙リンク条件

三辺 \(a,b,c\) が直角三角形を成すとは、

\[
a^2+b^2=c^2
\]

が成立することと定義する。

観測辺の定義を代入すると、

\[
\alpha^2 u_1^2+\beta^2 u_2^2=\gamma^2 u_3^2
\]

となる。

この式を ** 三単位宇宙リンク条件 ** と呼ぶ。

差分面積核 \(K_i=u_i^2\) を用いれば、

\[
\alpha^2 K_1+\beta^2 K_2=\gamma^2 K_3
\]

とも書ける。

---

三単位宇宙リンク条件のもとで、各辺に対応する差分量を

\[
\Delta_a := c^2-b^2,\qquad
\Delta_b := c^2-a^2,\qquad
\Delta_c := a^2+b^2
\]

と定義する。

これは、それぞれの辺平方が「どの差分量として生成されるか」を表す量である。

---

### 1.6. 境界差

斜辺 \(c\) と各短辺との差を

\[
\delta_{ca}:=c-a,\qquad
\delta_{cb}:=c-b
\]

と定義し、これを ** 境界差 ** と呼ぶ。

これは、ある辺から別の辺へ移る際の差分幅を表す。

---

### 1.7. 差分生成式

境界差を用いると、各短辺平方は

\[
b^2 = c^2-a^2,\qquad
a^2 = c^2-b^2
\]

と表される。

さらに

\[
c=a+\delta_{ca}
\]

を代入すると

\[
b^2=(a+\delta_{ca})^2-a^2
\]

であり、展開すると

\[
b^2=2a\delta_{ca}+\delta_{ca}^2
\]

となる。

同様に

\[
c=b+\delta_{cb}
\]

から

\[
a^2=2b\delta_{cb}+\delta_{cb}^2
\]

が得られる。

これらを ** 差分生成式 ** と呼ぶ。

\]

のような d 次冪宇宙リンク条件への一般化を検討する。

これにより、ピタゴラス構造を二次元固有のものとしてでなく、
** 差分核生成法則の特別な 2 次例 ** として位置づけ直せる可能性がある。

---

## 5. 暫定まとめ

本稿では、ピタゴラス構造を

\[
a^2+b^2=c^2
\]

という古典的関係式としてではなく、

\[
\alpha^2 u_1^2+\beta^2 u_2^2=\gamma^2 u_3^2
\]

という ** 三単位宇宙リンク条件 ** として再定義した。

この視点では、

- \(u_i\): 単位核
- \(u_i^2\): 差分面積核
- \(a,b,c\): 観測辺
- \(\Delta_a,\Delta_b,\Delta_c\): 辺対応差分量
- \(\delta_{ca},\delta_{cb}\): 境界差

が基本対象となる。

さらに、各短辺平方は

\[
b^2=2a\delta_{ca}+\delta_{ca}^2,\qquad
a^2=2b\delta_{cb}+\delta_{cb}^2
\]

として、境界差により生成される差分量として記述された。

したがって、ピタゴラスの定理は
** 三つの単位宇宙のあいだに成立する二次差分整合条件 **
として理解できる。

この定義・補題・予想・課題の整理は、今後の研究の最小土台として機能する。
````

````md
    edgeA α u₁ ^ 2 + edgeB β u₂ ^ 2 = edgeC γ u₃ ^ 2 := by
  dsimp [linkCondition] at h
  rw [edgeA_sq, edgeB_sq, edgeC_sq]
  simpa using h

/--
差分面積核を使った書き換え版。
-/
theorem linkCondition_iff_kernel
    (α β γ u₁ u₂ u₃ : R) :
    linkCondition α β γ u₁ u₂ u₃ ↔
      α ^ 2 * kernel u₁ + β ^ 2 * kernel u₂ = γ ^ 2 * kernel u₃ := by
  dsimp [linkCondition, kernel]

end DkMath.PythagorasCosmic

## 3. `DkMath/PythagorasCosmic/Delta.lean`

```lean
import DkMath.PythagorasCosmic.LinkCondition

namespace DkMath.PythagorasCosmic

variable {R : Type*} [LinearOrderedRing R]

/--
ピタゴラス条件のもとで、a に対応する差分量は a^2 に一致する。
-/
theorem deltaA_eq_a_sq
    {a b c : R}
    (h : a ^ 2 + b ^ 2 = c ^ 2) :
    deltaA a b c = a ^ 2 := by
  dsimp [deltaA]
  calc
    c ^ 2 - b ^ 2 = (a ^ 2 + b ^ 2) - b ^ 2 := by rw [← h]
    _ = a ^ 2 := by ring

/--
ピタゴラス条件のもとで、b に対応する差分量は b^2 に一致する。
-/
theorem deltaB_eq_b_sq
    {a b c : R}
    (h : a ^ 2 + b ^ 2 = c ^ 2) :
    deltaB a b c = b ^ 2 := by
  dsimp [deltaB]
  calc
    c ^ 2 - a ^ 2 = (a ^ 2 + b ^ 2) - a ^ 2 := by rw [← h]
    _ = b ^ 2 := by ring

/--
ピタゴラス条件のもとで、c に対応する差分量は c^2 に一致する。
-/
theorem deltaC_eq_c_sq
    {a b c : R}
    (h : a ^ 2 + b ^ 2 = c ^ 2) :
    deltaC a b c = c ^ 2 := by
  dsimp [deltaC]
  simpa [h]

/--
3 本まとめた版。
-/
theorem deltas_eq_sq
    {a b c : R}
    (h : a ^ 2 + b ^ 2 = c ^ 2) :
    deltaA a b c = a ^ 2 ∧
    deltaB a b c = b ^ 2 ∧
    deltaC a b c = c ^ 2 := by
  refine ⟨deltaA_eq_a_sq h, deltaB_eq_b_sq h, deltaC_eq_c_sq h⟩

/--
リンク条件から差分量の一致までを直接つなぐ版。
-/
theorem deltas_eq_sq_of_linkCondition
    (α β γ u₁ u₂ u₃ : R)
    (hlink : linkCondition α β γ u₁ u₂ u₃) :
    deltaA (edgeA α u₁) (edgeB β u₂) (edgeC γ u₃) = edgeA α u₁ ^ 2 ∧
    deltaB (edgeA α u₁) (edgeB β u₂) (edgeC γ u₃) = edgeB β u₂ ^ 2 ∧
    deltaC (edgeA α u₁) (edgeB β u₂) (edgeC γ u₃) = edgeC γ u₃ ^ 2 := by
  apply deltas_eq_sq
  exact pythagoras_of_linkCondition α β γ u₁ u₂ u₃ hlink

## 4. `DkMath/PythagorasCosmic/Boundary.lean`

```lean
import DkMath.PythagorasCosmic.Delta

namespace DkMath.PythagorasCosmic

variable {R : Type*} [LinearOrderedRing R]

/--
斜辺と a の境界差による差分展開。
\[
c^2 - a^2 = 2a(c-a) + (c-a)^2
\]
-/
theorem delta_from_boundaryCA
    (a c : R) :
    c ^ 2 - a ^ 2 = 2 * a * boundaryCA a c + boundaryCA a c ^ 2 := by
  dsimp [boundaryCA]
  ring

/--
斜辺と b の境界差による差分展開。
\[
c^2 - b^2 = 2b(c-b) + (c-b)^2
\]
-/
theorem delta_from_boundaryCB
    (b c : R) :
    c ^ 2 - b ^ 2 = 2 * b * boundaryCB b c + boundaryCB b c ^ 2 := by
  dsimp [boundaryCB]
  ring

/--
\(b^2 = c^2-a^2\) の仮定のもとで、b^2 を boundaryCA で表す。
-/
theorem short_leg_sq_from_boundaryCA
    {a b c : R}
    (h : b ^ 2 = c ^ 2 - a ^ 2) :
    b ^ 2 = 2 * a * boundaryCA a c + boundaryCA a c ^ 2 := by
  rw [h]
  exact delta_from_boundaryCA a c

/--
\(a^2 = c^2-b^2\) の仮定のもとで、a^2 を boundaryCB で表す。
-/
theorem short_leg_sq_from_boundaryCB
    {a b c : R}
    (h : a ^ 2 = c ^ 2 - b ^ 2) :
    a ^ 2 = 2 * b * boundaryCB b c + boundaryCB b c ^ 2 := by
  rw [h]
  exact delta_from_boundaryCB b c

/--
ピタゴラス条件から b^2 の境界差表示を得る。
-/
theorem short_leg_sq_from_pythagoras_CA
    {a b c : R}
    (h : a ^ 2 + b ^ 2 = c ^ 2) :
    b ^ 2 = 2 * a * boundaryCA a c + boundaryCA a c ^ 2 := by
  apply short_leg_sq_from_boundaryCA
  calc
    b ^ 2 = c ^ 2 - a ^ 2 := by
      linarith
    _ = c ^ 2 - a ^ 2 := rfl

/--
ピタゴラス条件から a^2 の境界差表示を得る。
-/
theorem short_leg_sq_from_pythagoras_CB
    {a b c : R}
    (h : a ^ 2 + b ^ 2 = c ^ 2) :
    a ^ 2 = 2 * b * boundaryCB b c + boundaryCB b c ^ 2 := by
  apply short_leg_sq_from_boundaryCB
  calc
    a ^ 2 = c ^ 2 - b ^ 2 := by
      linarith
    _ = c ^ 2 - b ^ 2 := rfl

/--
宇宙式型差分生成の抽象形。
-/
theorem sq_diff_eq_linear_plus_kernel
    (x u : R) :
    (x + u) ^ 2 - x ^ 2 = 2 * x * u + u ^ 2 := by
  ring

/--
boundaryCA は宇宙式型差分生成の具体例である。
-/
theorem boundaryCA_as_cosmic_form
    (a c : R) :
    c ^ 2 - a ^ 2 =
      2 * a * boundaryCA a c + kernel (boundaryCA a c) := by
  dsimp [kernel]
  exact delta_from_boundaryCA a c

/--
boundaryCB は宇宙式型差分生成の具体例である。
-/
theorem boundaryCB_as_cosmic_form
    (b c : R) :
    c ^ 2 - b ^ 2 =
      2 * b * boundaryCB b c + kernel (boundaryCB b c) := by
  dsimp [kernel]
  exact delta_from_boundaryCB b c

end DkMath.PythagorasCosmic
```

## 6. 補足

この 4 分割の利点は明快じゃ。

- `Basic` で ** 定義だけ **
- `LinkCondition` で ** 観測辺とリンク条件 **
- `Delta` で ** 差分量の一致 **
- `Boundary` で ** 宇宙式型差分生成 **

という層がはっきり分かれる。

特に `Boundary.lean` の

\[
c^2-a^2 = 2a(c-a) + (c-a)^2
\]

は、お主の研究ではかなり重要な橋になる。
ピタゴラス差分を「和の恒等式」ではなく ** 境界差生成式 ** として読む入口じゃからの。
````
