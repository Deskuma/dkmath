# 素因数分解の一意性を差の冪核比較へ輸送する方法

cid: 69c264eb-cc3c-83a7-98dd-f19e4b6cac7b

## 素数層分解・\(p\)-進付値・境界積を用いた証明構成

## 要旨

本稿では、自然数の素因数分解の一意性を、差の冪分解に現れる核多項式と境界積との比較へ輸送する証明方法を述べる。中心となるのは

\[
(x+u)^d-u^d =
x\,G_d(x,u)
\]

という分解であり、ここで

\[
G_d(x,u) =
\sum_{j=0}^{d-1}\binom{d}{j+1}x^j u^{d-1-j}
\]

を右核と呼ぶ。本稿では、境界積 \(xu\) と右核 \(G_d(x,u)\) の素因子構造を、素数 \(q\) が指数 \(d\) を割るか否かで二層に分けて比較する。例外層 \(q\mid d\) では係数消滅と \(p\)-進付値の一致が働き、非例外層 \(q\nmid d\) では非整除性と零付値化が働く。これにより、二つの自然数 \(m,n\) をそれぞれ \(xu\)、\(G_d(x,u)\) に比較した後、素因数分解の一意性により最終的に \(m=n\) を得る。

本稿の主眼は、素因数分解の一意性そのものの新証明ではなく、それを差の冪核に基づく証明系へ接続するための組み立て原理を明文化することにある。

## 1. はじめに

自然数の素因数分解の一意性は整数論の最も基本的な原理の一つである。しかし実際の形式的証明や構造的研究では、この一意性は単独で現れるよりも、より大きな証明の終端に置かれることが多い。すなわち、まずある二つの自然数について「すべての素数の寄与が一致する」ことを示し、その後に唯一分解によって自然数そのものの一致を回収する、という流れである。

本稿で扱う対象は、差の冪分解に現れる核

\[
G_d(x,u) =
\sum_{j=0}^{d-1}\binom{d}{j+1}x^j u^{d-1-j}
\]

である。この核は恒等式

\[
(x+u)^d-u^d =
x\,G_d(x,u)
\]

を満たし、差の冪を「境界」と「内部核」へ分解する役割を持つ。

ここで本稿は、右辺の境界 \(x\) だけではなく、左右の境界から作られる積

\[
B(x,u):=xu
\]

を導入し、これと \(G_d(x,u)\) との比較を通じて、他の自然数 \(m,n\) の一致を示す方法を整理する。

基本構図は

\[
m
\longleftrightarrow
B(x,u)
\longleftrightarrow
G_d(x,u)
\longleftrightarrow
n
\]

という比較の鎖である。ここで中央の橋 \(B(x,u)\leftrightarrow G_d(x,u)\) は、素数が指数 \(d\) を割るかどうかで性質が異なるため、素数を二層に分けて扱う必要がある。

## 2. 基本定義

### 2.1. \(p\)-進付値

自然数 \(N\neq 0\) と素数 \(p\) に対し、\(p\)-進付値 \(v_p(N)\) を

\[
p^{v_p(N)}\mid N,
\qquad
p^{v_p(N)+1}\nmid N
\]

によって定める。

これにより、任意の \(k\in\mathbb N\) に対し

\[
p^k\mid N
\iff
k\le v_p(N)
\]

が成り立つ。

### 2.2. 素数支持

自然数 \(N\neq 0\) の素数支持を

\[
\operatorname{Supp}(N) =
\{\,p \mid p \text{ は素数で } p\mid N\,\}
\]

と定める。

### 2.3. 境界と核

自然数 \(x,u,d\) に対して、以下の量を考える。

右境界：

\[
R(x,u):=x
\]

左境界：

\[
L(x,u):=u
\]

境界積：

\[
B(x,u):=xu
\]

右核：

\[
G_d(x,u) =
\sum_{j=0}^{d-1}\binom{d}{j+1}x^j u^{d-1-j}
\]

左核：

\[
G_d(u,x) =
\sum_{j=0}^{d-1}\binom{d}{j+1}u^j x^{d-1-j}
\]

本稿では主に \(B(x,u)\) と \(G_d(x,u)\) を扱う。

### 2.4. 例外層と非例外層

素数 \(q\) に対して、\(q\mid d\) ならば \(q\) を例外層の素数と呼び、\(q\nmid d\) ならば \(q\) を非例外層の素数と呼ぶ。

この区別は本稿の中心である。例外層では \(d\) に由来する係数が \(q\) で消滅し、非例外層ではその消滅が起こらないため、比較の機構が本質的に異なる。

## 3. 素因数分解の一意性の基本形

本稿の基礎となるのは、素数冪整除の一致から自然数の一致を得る以下の定理である。

### 定理 3.1

自然数 \(m,n\neq 0\) に対し、任意の素数 \(p\) と任意の \(k\in\mathbb N\) について

\[
p^k\mid m
\iff
p^k\mid n
\]

が成り立つならば、

\[
m=n
\]

である。

### 証明

仮定より、任意の素数 \(p\) と任意の \(k\) に対し

\[
k\le v_p(m)
\iff
k\le v_p(n)
\]

が成り立つ。したがって

\[
v_p(m)=v_p(n)
\]

がすべての素数 \(p\) について従う。ゆえに素因数分解の指数がすべて一致し、

\[
m=\prod_p p^{v_p(m)} =
\prod_p p^{v_p(n)} =
n
\]

となる。 \(\square\)

この定理は本稿全体の終端に位置する。以下では、\(m\) と \(n\) の素数冪整除の一致を直接示す代わりに、それぞれを \(B(x,u)\) と \(G_d(x,u)\) に比較することで間接的に証明する。

## 4. 素数支持に関する準備

素数支持の一致は、素数整除の一致から従う。

### 命題 4.1

自然数 \(N\neq 0\) と素数 \(p\) について

\[
p\in \operatorname{Supp}(N)
\iff
p\mid N
\]

が成り立つ。

### 証明

素因数分解において \(p\) が現れることと、\(p\) が \(N\) を割ることは同値である。 \(\square\)

### 命題 4.2

自然数 \(m,n\neq 0\) に対し、任意の素数 \(p\) について

\[
p\mid m
\iff
p\mid n
\]

が成り立つならば

\[
\operatorname{Supp}(m)=\operatorname{Supp}(n)
\]

である。

### 証明

命題 4.1 を各素数 \(p\) に適用すればただちに従う。 \(\square\)

この準備は、後に「どの素数がそもそも現れるか」という粗い比較と、「各素数が何回現れるか」という細かい比較を分けて考える際に有用である。

## 5. 例外層における核への流入

例外層 \(q\mid d\) では、右核の先頭係数が \(d\) を含むため、右境界の整除が核へ流入する。

### 命題 5.1

\(d\ge 1\) とし、\(q\) を素数とする。もし

\[
q\mid d,
\qquad
q\mid x
\]

ならば

\[
q\mid G_d(x,u)
\]

である。

### 証明

\(x\equiv 0 \pmod q\) のもとで、核の各項のうち \(x\) を含む項はすべて \(q\) で消える。したがって

\[
G_d(x,u)
\equiv
\binom d1 u^{d-1} =
d\,u^{d-1}
\pmod q
\]

を得る。ところが \(q\mid d\) であるから右辺は \(0\) に合同である。よって

\[
G_d(x,u)\equiv 0\pmod q
\]

すなわち

\[
q\mid G_d(x,u)
\]

である。 \(\square\)

### 命題 5.2

\(d\ge 1\) とし、\(q\) を素数とする。もし

\[
q\mid d,
\qquad
q\mid xu,
\qquad
q\nmid u
\]

ならば

\[
q\mid G_d(x,u)
\]

である。

### 証明

\(q\mid xu\) かつ \(q\nmid u\) より \(q\mid x\) である。したがって命題 5.1 を適用すればよい。 \(\square\)

この結果は、例外層では境界積側の素数が核側へ流れ込むことを意味する。さらに \(p\)-進付値が一致していれば、単なる整除だけでなく全ての素数冪整除が一致する。

### 命題 5.3

\(q\) を例外層の素数とし、

\[
v_q(B(x,u)) =
v_q(G_d(x,u))
\]

と仮定する。このとき任意の \(k\in\mathbb N\) に対して

\[
q^k\mid B(x,u)
\iff
q^k\mid G_d(x,u)
\]

が成り立つ。

### 証明

\[
q^k\mid N
\iff
k\le v_q(N)
\]

を \(N=B(x,u)\) および \(N=G_d(x,u)\) に適用し、付値の等式を用いればよい。 \(\square\)

### 命題 5.4

例外層において、もし

\[
v_q(B(x,u))
\le
v_q(G_d(x,u))
\]

が成り立つならば

\[
q^k\mid B(x,u)
\Longrightarrow
q^k\mid G_d(x,u)
\]

であり、逆向き不等式

\[
v_q(G_d(x,u))
\le
v_q(B(x,u))
\]

が成り立つならば

\[
q^k\mid G_d(x,u)
\Longrightarrow
q^k\mid B(x,u)
\]

である。

### 証明

いずれも

\[
q^k\mid N
\iff
k\le v_q(N)
\]

を用いれば直ちに従う。両向きの不等式があれば付値は一致し、命題 5.3 に帰着する。 \(\square\)

以上より、例外層の本質は

\[
\text{係数消滅}
\quad+\quad
\text{付値比較}
\]

にある。

## 6. 非例外層における零付値化

非例外層 \(q\nmid d\) では、例外層とは逆に、境界積側と核側の素数寄与を消す方向が自然である。

### 命題 6.1

\(q\) を非例外層の素数とする。もし

\[
q\nmid B(x,u),
\qquad
q\nmid G_d(x,u)
\]

ならば

\[
v_q(B(x,u))=0,
\qquad
v_q(G_d(x,u))=0
\]

である。

### 証明

素数 \(q\) が自然数 \(N\) を割らなければ \(v_q(N)=0\) である。これをそれぞれに適用する。 \(\square\)

### 命題 6.2

\(q\) を非例外層の素数とし、

\[
v_q(B(x,u))=0,
\qquad
v_q(G_d(x,u))=0
\]

と仮定する。このとき任意の \(k\in\mathbb N\) に対して

\[
q^k\mid B(x,u)
\iff
q^k\mid G_d(x,u)
\]

が成り立つ。

### 証明

両辺とも \(k=0\) のときのみ真であり、\(k>0\) ではいずれも偽である。あるいは

\[
q^k\mid N
\iff
k\le v_q(N)
\]

に \(v_q(N)=0\) を代入してもよい。 \(\square\)

### 命題 6.3

もし

\[
v_q(G_d(x,u))
\le
v_q(B(x,u))
\]

であり、かつ \(q\nmid B(x,u)\) ならば \(q\nmid G_d(x,u)\) である。

### 証明

\(q\nmid B(x,u)\) より \(v_q(B(x,u))=0\) である。したがって

\[
v_q(G_d(x,u))
\le
0
\]

となり、自然数値であることから

\[
v_q(G_d(x,u))=0
\]

である。ゆえに \(q\nmid G_d(x,u)\)。 \(\square\)

この命題は非例外層における「弱い核条件」を表している。すなわち、核側の非整除を直接仮定しなくても、境界積側の非整除と逆向き付値不等式だけで核側の非整除が導ける。

## 7. 境界側入力の統一

非例外層で境界積側の非整除を得る方法は二通りある。

第一は直接

\[
q\nmid xu
\]

を与える方法である。第二は左右の境界に対して

\[
q\nmid x,
\qquad
q\nmid u
\]

を与える方法である。この二つは次の形で統一できる。

### 命題 7.1

もし

\[
q\nmid x,
\qquad
q\nmid u
\]

ならば

\[
q\nmid xu
\]

である。

### 証明

もし \(q\mid xu\) なら素数性より \(q\mid x\) または \(q\mid u\) である。仮定に反する。 \(\square\)

したがって、境界積側の非整除条件は

\[
q\nmid xu
\]

を直接置いてもよいし、左右境界の非整除

\[
q\nmid x,
\qquad
q\nmid u
\]

を置いてもよい。形式化の観点では、この二つを同じ入口として扱えることが重要である。

## 8. 橋条件の統一

境界積 \(B(x,u)\) と右核 \(G_d(x,u)\) の比較には、やはり二つの入口がある。

一つは \(p\)-進付値の等式である。すなわち

\[
v_q(B(x,u)) =
v_q(G_d(x,u))
\]

を与える方法である。もう一つは、はじめから全ての \(k\) に対する素数冪整除同値

\[
q^k\mid B(x,u)
\iff
q^k\mid G_d(x,u)
\]

を与える方法である。

付値等式があれば前者から後者が導かれる。したがって橋条件の入力としては、付値等式でも素数冪整除同値でも本質的には同じ働きをする。

この観点に立つと、証明の設計は

1. 付値で比較するか
2. 整除同値で比較するか

のどちらかを選ぶだけでよい。

## 9. 層別比較から自然数の一致へ

ここで二つの自然数 \(m,n\neq 0\) をとる。目標は \(m=n\) を示すことである。そのために、例外層と非例外層で別々に比較を用意する。

### 定理 9.1. 層別比較による因数分解一致

\(d,m,n\) を自然数とし、\(m,n\neq 0\) とする。次を仮定する。

任意の例外層素数 \(q\mid d\) と任意の \(k\) に対して

\[
q^k\mid m
\iff
q^k\mid n
\]

が成り立つ。

また、任意の非例外層素数 \(q\nmid d\) と任意の \(k\) に対して

\[
q^k\mid m
\iff
q^k\mid n
\]

が成り立つ。

このとき

\[
m=n
\]

である。

### 証明

任意の素数 \(q\) は \(q\mid d\) または \(q\nmid d\) のいずれかである。したがって仮定により、全ての素数 \(q\) と全ての \(k\) に対し

\[
q^k\mid m
\iff
q^k\mid n
\]

が成り立つ。よって定理 3.1 により \(m=n\) である。 \(\square\)

この定理は見た目には自明であるが、証明を「例外層」と「非例外層」に分割できることを保証するという意味で重要である。

## 10. 境界積と右核を介した主定理

次に、本稿の中心的な end-to-end 定理を述べる。

### 定理 10.1. 境界積・右核比較による end-to-end 一致定理

自然数 \(d,x,u,m,n\) が

\[
d\ge 2,
\qquad
x>0,
\qquad
u>0,
\qquad
m\neq 0,
\qquad
n\neq 0
\]

を満たしているとする。さらに以下を仮定する。

#### 例外層の仮定

任意の素数 \(q\mid d\) と任意の \(k\) に対し、

\[
q^k\mid m
\iff
q^k\mid B(x,u),
\]

\[
q^k\mid n
\iff
q^k\mid G_d(x,u),
\]

\[
q^k\mid B(x,u)
\iff
q^k\mid G_d(x,u)
\]

が成り立つ。

#### 非例外層の仮定

任意の素数 \(q\nmid d\) と任意の \(k\) に対し、

\[
q^k\mid m
\iff
q^k\mid B(x,u),
\]

\[
q^k\mid n
\iff
q^k\mid G_d(x,u),
\]

\[
q^k\mid B(x,u)
\iff
q^k\mid G_d(x,u)
\]

が成り立つ。

このとき

\[
m=n
\]

である。

### 証明

例外層では推移律により

\[
q^k\mid m
\iff
q^k\mid n
\]

が成り立つ。非例外層でも同様である。したがって全ての素数 \(q\) と全ての \(k\) に対して

\[
q^k\mid m
\iff
q^k\mid n
\]

が成立するから、定理 3.1 により \(m=n\) である。 \(\square\)

## 11. 自動供給型の主定理

実際には、定理 10.1 の中央比較

\[
q^k\mid B(x,u)
\iff
q^k\mid G_d(x,u)
\]

を直接証明するよりも、付値条件から自動的に供給する方が自然である。その代表形を二つ述べる。

### 定理 11.1. 付値等式型

定理 10.1 の仮定のうち、中央比較を直接仮定する代わりに、例外層と非例外層のすべての素数 \(q\) について

\[
v_q(B(x,u)) =
v_q(G_d(x,u))
\]

を仮定してよい。このとき

\[
m=n
\]

である。

### 証明

付値等式から各層で

\[
q^k\mid B(x,u)
\iff
q^k\mid G_d(x,u)
\]

が従うので、定理 10.1 を適用すればよい。 \(\square\)

### 定理 11.2. 例外層は両向き不等式、非例外層は零付値型

定理 10.1 の仮定のうち、中央比較の代わりに次を仮定してよい。

例外層では、任意の \(q\mid d\) について

\[
v_q(B(x,u))
\le
v_q(G_d(x,u)),
\qquad
v_q(G_d(x,u))
\le
v_q(B(x,u)).
\]

非例外層では、任意の \(q\nmid d\) について

\[
v_q(B(x,u))=0,
\qquad
v_q(G_d(x,u))=0.
\]

このとき

\[
m=n
\]

である。

### 証明

例外層では二つの不等式から付値等式が従い、したがって素数冪整除同値が従う。非例外層では零付値から素数冪整除同値が従う。ゆえに定理 10.1 を適用すればよい。 \(\square\)

この形は、例外層を「比較」、非例外層を「消滅」で処理するという本稿の思想を最もよく表している。

## 12. 弱い核条件による非例外層処理

非例外層では、核側の非整除を直接仮定しなくてもよい場合がある。境界積側の非整除と逆向き付値不等式から核側の非整除が導かれるからである。

### 定理 12.1

自然数 \(d,x,u\) が

\[
d\ge 2,
\qquad
x>0,
\qquad
u>0
\]

を満たすとする。非例外層の任意の素数 \(q\nmid d\) について

\[
v_q(G_d(x,u))
\le
v_q(B(x,u))
\]

および

\[
q\nmid B(x,u)
\]

を仮定する。このとき

\[
q\nmid G_d(x,u)
\]

が成り立つ。

### 証明

命題 6.3 に一致する。 \(\square\)

### 系 12.2

上の仮定のもとで、非例外層では

\[
v_q(B(x,u))=0,
\qquad
v_q(G_d(x,u))=0
\]

が成り立つ。

### 証明

境界積側は非整除より零付値、核側は定理 12.1 により非整除、したがって零付値である。 \(\square\)

これにより、非例外層の核側に対して強い情報を直接求めなくてもよい。境界積側の非整除と、核から境界積への逆向き付値比較さえあれば十分である。

## 13. 証明戦略の全体像

本稿の証明方法は次のように整理できる。

### 13.1. 第一步

二つの自然数 \(m,n\) の一致を直接示すのではなく、全ての素数冪整除判定の一致へ還元する。

### 13.2. 第二歩

素数を例外層 \(q\mid d\) と非例外層 \(q\nmid d\) に分ける。

### 13.3. 第三歩

例外層では、右境界や境界積から右核への整除、および \(p\)-進付値の一致または両向き不等式を用いて、境界積と右核の素数冪整除を一致させる。

### 13.4. 第四歩

非例外層では、境界積側の非整除、および必要に応じて逆向き付値不等式を用いて、境界積と右核の付値をともに零にする。

### 13.5. 第五歩

\(m\) と境界積、\(n\) と右核の比較をそれぞれの層で与え、推移律によって \(m\) と \(n\) の素数冪整除を一致させる。

### 13.6. 第六歩

最後に素因数分解の一意性により \(m=n\) を得る。

この構造を見ると、唯一分解は「証明の出発点」ではなく、「層別比較を自然数の等式へ回収する終端装置」として働いていることが分かる。

## 14. 本手法の意義

本稿の手法には三つの意義がある。

第一に、差の冪分解に現れる核 \(G_d(x,u)\) を、単なる補助多項式ではなく、素因子比較の中心対象として位置づけた点である。

第二に、素数 \(q\) が指数 \(d\) を割るかどうかで比較機構が変わることを明示し、例外層と非例外層という二層構造に整理した点である。例外層では係数消滅が、非例外層では零付値化が主役になる。

第三に、証明の入口を柔軟にした点である。境界側については「境界積の非整除」と「左右境界の非整除」を同じ入口として扱うことができ、橋条件については「付値等式」と「素数冪整除同値」を同じ入口として扱うことができる。これにより、実際の応用場面では手元にある補題の形に応じて最適な入口を選択できる。

## 15. 結論

本稿では、素因数分解の一意性を、差の冪分解

\[
(x+u)^d-u^d=x\,G_d(x,u)
\]

に基づく境界積 \(B(x,u)=xu\) と右核 \(G_d(x,u)\) の比較へ輸送する証明方法を述べた。

中心思想は次の三点に要約できる。

\[
\text{唯一分解}
\quad+\quad
\text{素数の二層分解}
\quad+\quad
\text{境界積と核の } p\text{-進比較}
\]

である。

この方法により、複雑な整数論的主張を、例外層と非例外層の局所的比較へ分解し、最後に唯一分解で統合することが可能になる。したがって本稿の枠組みは、差の冪、原始素因子、付値評価、最大公約数制御などを同一の証明骨格の中で扱うための基盤として理解されるべきである。

## 16. 今後の見通し

本稿で述べた構成は、右核 \(G_d(x,u)\) を介した比較の一般骨格である。今後の発展としては、少なくとも次の三方向が考えられる。

第一に、例外層においては、単なる付値等式ではなく、核の先頭項や二項係数の局所的消滅をさらに精密化し、どの程度まで境界から核への流入が一般化できるかを調べる方向である。

第二に、非例外層においては、零付値化を導くための非整除条件を、より構造的な最大公約数制御や原始素因子条件へと置き換える方向である。

第三に、全体の end-to-end 構成を、差の冪方程式やフェルマー型方程式に現れる具体的な自然数 \(m,n\) に接続し、境界積と右核との比較が実際にどのような反証機構を生むかを調べる方向である。

これらはいずれも、本稿の証明骨格を土台として自然に展開される。
