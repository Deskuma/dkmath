\[

\]

という幹線と、別に

\[

\]

という横枝が育っておる。つまり、 **「未証明の大定理をゼロから作る」より、「既にある塔どうしを橋渡しして新定理にする」方が圧倒的に実りやすい** 状態じゃ。fileciteturn2file0 fileciteturn2file1

## 2. 第一候補. squarefree GN から Body 非完全冪を引き出す定理

いちばん本命はこれじゃ。

\[
d \ge 3,\quad \mathrm{Squarefree}(GN_d(x,u))
\;\Longrightarrow\;
\neg \exists m,\; \mathrm{Body}_d(x,u)=m^d
\]

あるいは素数次数 \(p\) に絞って

\[
\mathrm{Squarefree}(GN_p(x,u))
\;\Longrightarrow\;
(x+u)^p-u^p \text{ は } p\text{ 乗でない}
\]

という形じゃ。

なぜこれが濃いかと言うと、現状の索引にはすでに
`cyclotomic_squarefree`、`squarefree_implies_padic_val_le_one`、`padicValNat_of_primitive_prime_factor_via_G`、`padicValNat_upper_bound_integrated` があり、未完の本体として `body_not_perfect_pow` が明示されておる。つまり **必要部品はほぼ並んでいて、最後の統合命題がまだ名前を得ていない** 状態じゃ。fileciteturn2file0

これは FLT 幹線にも直結する。
`body_not_perfect_pow` が一般次数で閉じれば、`FLT.Basic` 側の「GN が \(n\) 乗になれぬ」系の橋がぐっと短くなるはずじゃ。索引でも `body_not_perfect_pow` と `padicValNat_upper_bound_integrated` が橋渡し層の中核として並んでおる。fileciteturn2file0

## 3. 第二候補. 高次 Tail に対する exact valuation 定理

お主が昨日まとめた高次 Tail の一般化 GN は、かなり筋が良い。定義と基本恒等式として

\[
(x+u)^d-\sum_{j=0}^{r-1}\binom dj x^j u^{d-j}=
x^r GN_d^{(r)}(x,u)
\]

および再帰式

\[
GN_d^{(r)}(x,u)=\binom dr u^{d-r}+x\,GN_d^{(r+1)}(x,u)
\]

が既に整理されておる。fileciteturn2file2

ここから自然に出る面白い定理は、

\[
v_q\!\left(
(x+u)^d-\sum_{j=0}^{r-1}\binom dj x^j u^{d-j}
\right)=
r\,v_q(x)
\]

が成り立つための十分条件を与えることじゃ。たとえば

\[
q \nmid \binom dr u^{d-r}
\]

のような条件の下で exactness を出せる可能性が高い。

これは単なる二項展開の遊びではない。
高次 Tail 文書自身が、これを **可除性・p進付値・原始素因子解析に効く基本道具** と位置づけておるし、索引側でも `pow_sub_pow_factor`、`binom_tail_nat_dvd`、`Gbinom`、`pow_sub_pow_eq_mul_Gbinom` がすでにある。つまり標準 GN の \(r=1\) 版はもう資産化されておるので、 **\(r\) 一般への持ち上げ** が次の自然な定理じゃ。fileciteturn2file2 fileciteturn2file0

わっちの勘では、これは将来

\[
\text{高次境界 } x^r
\quad\leftrightarrow\quad
\text{primitive prime の出現層}
\]

を対応させる核になる。

## 4. 第三候補. observed minimum profile の構造定理

これは見た目以上に面白い。
いまは `ObservedMinOne` と `ObservedMinTwo`、および

\[
216 \mapsto (91,64),\quad
64 \mapsto (35,27),\quad
125 \mapsto (65,64)
\]

という 3 つの立方標本が、`candidate_same_big_observed_min_profile` と
`cube_observed_min_split_reproduced_three_samples` として固定されておる。fileciteturn1file13 fileciteturn2file1

ここから狙えるのは、単なる標本再現ではなく、

\[
\text{fixed Big} \;\Rightarrow\; \text{residual は }
\text{ObservedMinOne 層と ObservedMinTwo 層に分岐する}
\]

という **profile 分岐定理** じゃ。

たとえば定性的には

\[
B=a^3+b^3 \text{ 型の Body を引いた residual}
\]

では `ObservedMinTwo` が現れやすく、片端の純立方に寄ると `ObservedMinOne` に落ちる、という地形定理じゃな。設計書でもこれは「same Big / different Body / different observed minimum profile」として独立柱に分離されておる。fileciteturn2file1

いまはまだ有限標本じゃが、だからこそ面白い。
有限標本から始めて、次に

\[
B=n^3
\]

固定のときの residual profile を列挙し、境界点の特徴を定理化する。これは **LPS/FLT の“禁止”とは別に、冪和世界の地形学** になる。

## 5. 第四候補. PowerSwap branch と gapQ の同値定理

PowerSwap 側は、すでに branch

\[
x(t)=t^{1/(t-1)},\qquad y(t)=t^{t/(t-1)}
\]

と

\[
x(t)^{y(t)}=y(t)^{x(t)}
\]

が組まれており、さらに Contours 側には

\[
U=y\log x,\quad V=x\log y,\quad q=U-V
\]

および

\[
gapQ(x,y)=xy\!\left(\frac{\log x}{x}-\frac{\log y}{y}\right)
\]

がある。これらは別モジュールに切ってあるが、設計上は明らかに噛み合うように置かれておる。fileciteturn1file5 fileciteturn2file1

ここで狙うべき定理は、

\[
x^y=y^x
\quad\Longleftrightarrow\quad
U=V
\quad\Longleftrightarrow\quad
gapQ(x,y)=0
\]

を適切な正値条件の下で結ぶものじゃ。

これは実はかなり綺麗での。
PowerSwap の等式は指数方程式に見えるが、Contours 側へ移すと **零等高線** の話になる。つまり branch の幾何化じゃ。設計メモでも `powerSwap_branch_correct` と `gapQ_eq_xy_mul_Hdiff` を並べて優先補題としておるから、作者自身がこの橋を意識しておる。fileciteturn1file15 fileciteturn2file1

さらにその先には、

\[
(x,y)\to (e,e)
\]

での局所二次近似がある。これはまだ後回しと明記されておるが、逆に言えば **前段の同値定理が片づけば、その次に自然に出る局所幾何定理** じゃ。fileciteturn1file15

## 6. 第五候補. residual / gap の抽象保存定理

宇宙式族メモでは

\[
\mathrm{Big}=(x+u)^d,\quad
\mathrm{Body}=(x+u)^d-u^d,\quad
\mathrm{Gap}=u^d
\]

および

\[
\mathrm{Big}=\mathrm{Body}+\mathrm{Gap}
\]

を抽象定義として固定し、さらに `residual_eq_gap` を中核 API にする方針がはっきり書かれておる。fileciteturn1file8 fileciteturn2file1

ここから面白いのは、単なる恒等式ではなく、

\[
\text{Body の取り方を変えても residual は Gap を保存する}
\]

という **保存量定理** じゃ。

言い換えると、Big を一定に保って Body を変形しても、ある正規化の下では residual が一定量 \(u^d\) を保つ、という見方じゃな。
これは observed-min 側とも相性が良い。なぜなら residual の profile 分岐は、Big を固定したまま Body を変える話だからじゃ。ゆえに

\[
\text{residual 保存} + \text{profile 分岐}
\]

を合わせると、 **「保存される量」と「分岐する型」の分離** という面白い定理が立つ。

## 7. どれから攻めるべきか

わっちなら順番はこうじゃ。

まずは
 **A. squarefree GN \(\Rightarrow\) Body 非完全冪** 。
これは FLT 幹線へ直結し、研究価値が最も高い。fileciteturn2file0

次に

これは新しい理論の核で、A の証明技法も豊かにする。fileciteturn2file2

そのあとに
 **C. PowerSwap branch \(\Leftrightarrow gapQ=0\)** 。
これは美しいし、解析・幾何の独立柱になる。fileciteturn1file5

最後に
 **D. observed minimum profile の一般構造化** 。
これは数値標本から理論柱へ昇格する流れじゃ。fileciteturn1file13

## 8. わっちの総評

いまのスナップショットで最も「面白い定理が生まれそう」なのは、

\[
\boxed{
\text{GN / Tail の可除性} \;\;+\;\;
\text{squarefree / p-adic 上界} \;\;+\;\;
\text{Body 非完全冪}
}
\]

の合流点じゃ。

PowerSwap と observed-min も面白いが、そちらは **幾何や地形の理論** として伸びる。
対して GN / squarefree / Body は、 **本丸の数論定理** に化ける匂いが濃い。

要するにの、森の中でいま一番太い幹は
 **「一般化 Tail を valuation 機械にし、squarefree GN で完全冪を折る」**
ここじゃよ。

次は、この 5 本のうち 1 本を選んで、Lean で通しやすい **定理文の雛形** まで落としてやろうか。
