妙案、ある。これは **Bézout 恒等式** として読むのが一番きれいじゃ。

\[
(X+1)^2-X(X+2)=1
\]

これは単なる差分恒等式ではなく、環 \(\mathbb Z[X]\) において

\[
(X(X+2),\,(X+1)^2)=\mathbb Z[X]
\]

つまり、二つの多項式 \(X(X+2)\) と \((X+1)^2\) は **互いに素** である、という証明そのものになっておる。

整数 \(x\in\mathbb N\) に評価すると、

\[
(x+1)^2-x(x+2)=1
\]

ゆえに任意の共通因子 \(d\) が

\[
d\mid x(x+2),\qquad d\mid (x+1)^2
\]

を満たすなら、その差にも割り込むので、

\[
d\mid \bigl((x+1)^2-x(x+2)\bigr)=1
\]

したがって

\[
\gcd(x(x+2),(x+1)^2)=1
\]

じゃ。

つまり、正当な橋はこう。

\[
N_x=x(x+2)
\]

と置けば、

\[
N_x+1=(x+1)^2
\]

なので、

\[
\gcd(N_x,N_x+1)=\gcd(x(x+2),(x+1)^2)=1
\]

となる。

だから

\[
\gcd(N,N+1)=f(x)
\]

と直接同一視するより、

\[
f(x)=1
\]

は

\[
N_x=x(x+2),\qquad N_x+1=(x+1)^2
\]

という **宇宙式型の連続整数ペアが互いに素であることの Bézout 証明書** と読むのがよい。

もう少し構造的に言えば、

\[
(x+1)^2-1=x(x+2)
\]

は

\[
y=x+1
\]

と置いて、

\[
y^2-1=(y-1)(y+1)
\]

つまり **中心平方 \(y^2\)** と **その左右隣接積 \((y-1)(y+1)\)** の差が常に \(1\) である、という構造じゃ。

よってこれは、

\[
\text{中心平方} - \text{左右隣接積} = 1
\]

であり、同時に

\[
\text{中心平方と左右隣接積は互いに素}
\]

を保証する。

わっちならこう名付けるかのぅ。

\[
\boxed{\text{宇宙式は、隣接積と中心平方の Bézout 境界である}}
\]

可除性の \(1\) と差分の \(1\) は意味が違う。だが、 **差分の \(1\) が、可除性の \(1\) を証明している** 。ここが正当な接続じゃ。

通常は

\[
\gcd(a,b)=1
\]

を

\[
a,b \text{ の共通因子は単元だけ}
\]

と読む。整数環 \(\mathbb Z\) の単元は \(\pm 1\) だけなので、自然数側では実質 \(1\) だけじゃ。

ここで、基底単位として許す素数集合を \(U\) とする。たとえば \(U=\{3\}\) や \(U=\{2,3\}\) じゃな。

\[
S_U=\left\{\prod_{p\in U}p^{e_p}\mid e_p\in\mathbb N\right\}
\]

と置く。これは \(U\) の素数だけから作られる数全体。これを **単位化してよい共通単位族** と見る。

すると、ぬしの言いたい性質はこう書ける。

\[
a \perp_U b
\quad\Longleftrightarrow\quad
\gcd(a,b)\in S_U
\]

つまり、

\[
\text{共通因子はある。だが、それは基底単位だけから成る。}
\]

さらに言えば、

\[
\text{基底単位を取り除いた後では、}a,b\text{ は互いに素。}
\]

という意味になる。

たとえば \(U=\{3\}\) として、

\[
a=15=3\cdot 5,\qquad b=21=3\cdot 7
\]

なら、

\[
\gcd(15,21)=3
\]

なので普通には互いに素ではない。だが \(3\) を基底単位として単元化するなら、

\[
15 \sim 5,\qquad 21 \sim 7
\]

であり、

\[
\gcd(5,7)=1
\]

となる。したがって

\[
15\perp_{\{3\}}21
\]

と読める。

これがまさに、ぬしの言う

> ２つの \(a,b\) の共通単位。これは、どっちも割る。だけど、その単位からの整数では \(a,b\) を割る因子は存在しない。

じゃ。

より構造的には、環を

\[
\mathbb Z \longrightarrow S_U^{-1}\mathbb Z
\]

へ移す。これは \(U\) の素数を逆元化する操作じゃ。たとえば \(U=\{3\}\) なら、

\[
\mathbb Z[1/3]
\]

になる。この世界では \(3\) は単元なので、\(3\) による共通因子は「共通因子として数えない」。

だから正当な一般化はこれじゃ。

\[
\boxed{
a,b\text{ は }U\text{-局所的に互いに素}
\quad\Longleftrightarrow\quad
\gcd(a,b)\text{ は }U\text{-単位}
}
\]

あるいは、より Lean 実装向けに言うなら、

\[
\operatorname{core}_U(n) =
\frac{n}{\prod_{p\in U}p^{v_p(n)}}
\]

として、基底単位成分を削った核を定義し、

\[
\boxed{
a\perp_U b
\quad\Longleftrightarrow\quad
\gcd(\operatorname{core}_U(a),\operatorname{core}_U(b))=1
}
\]

とするのが扱いやすい。

通常の互いに素は、\(U=\varnothing\) の特殊ケースじゃ。

\[
a\perp_{\varnothing} b
\quad\Longleftrightarrow\quad
\gcd(a,b)=1
\]

逆に、\(U\) を大きくすれば、より多くの共通因子を「基底単位として無視」できる。

を与える。これは \(U=\varnothing\) の最強形じゃ。

## 4. 局所化解釈

\(u\) を単位化する局所化環を考える。

\[
\mathbb Z[1/u]
\]

この世界では \(u\) は単元になる。
したがって \(u^2\) も単元じゃ。

宇宙式より、

\[
A-B=u^2
\]

なので、イデアル \((A,B)\) は \(u^2\) を含む。

\[
u^2\in (A,B)
\]

局所化後には \(u^2\) が単元なので、

\[
(A,B)=\mathbb Z[1/u]
\]

となる。

つまり、

\[
\boxed{
\mathbb Z[1/u]\text{ 上で }A\text{ と }B\text{ は comaximal}
}
\]

じゃ。

---

## 5. イデアル解釈

整数環 \(\mathbb Z\) では、

\[
u^2\in \langle A,B\rangle
\]

が成り立つ。

局所化後には、

\[
\langle A,B\rangle=\top
\]

となる。

すなわち、

\[
\boxed{
\text{宇宙式は、局所化後の comaximal 証明書である}
}
\]

と言える。

これは gcd の話を、イデアルの話へ昇格した形じゃな。

## 6. 素数チャネル解釈

任意の素数 \(p\) について、

\[
p\nmid u
\]

なら、\(p\) は \(A\) と \(B\) を同時には割れない。

なぜなら、もし

\[
p\mid A,\qquad p\mid B
\]

なら、

\[
p\mid A-B=u^2
\]

となり、

\[
p\mid u
\]

に矛盾するからじゃ。

したがって、

\[
\boxed{
p\nmid u
\Longrightarrow
\neg(p\mid A\land p\mid B)
}
\]

これは Petal 的には、

\[
\text{基底単位外の素数は、Big と Core の共通チャネルに入れない}
\]

という意味になる。

## 7. 宇宙式から言える主張

まとめると、宇宙式はこう読める。

\[
(x+u)^2=x(x+2u)+u^2
\]

これは、

\[
\text{Big}=\text{Core}+\text{UnitGap}
\]

であり、同時に、

\[
\text{Big と Core の共通因子は UnitGap に閉じ込められる}
\]

という主張である。

よって、

\[
u\text{ を基底単位として単位化すれば、Big と Core は互いに素}
\]

となる。

一言でまとめるなら、これじゃ。

\[
\boxed{
\text{宇宙式は、基底単位を除いた世界での互素性を主張している}
}
\]

## 9. 最終整理

通常の宇宙式：

\[
(x+1)^2-x(x+2)=1
\]

これは普通の互素性。

\[
\gcd\bigl((x+1)^2,\;x(x+2)\bigr)=1
\]

一般単位版：

\[
(x+u)^2-x(x+2u)=u^2
\]

これは基底単位互素性。

\[
\gcd\bigl((x+u)^2,\;x(x+2u)\bigr)\mid u^2
\]

局所化後：

\[
\langle (x+u)^2,\;x(x+2u)\rangle=\top
\quad\text{in }\mathbb Z[1/u]
\]

つまり、

\[
\boxed{
\text{宇宙式}
\Rightarrow
\text{Bézout}
\Rightarrow
\text{gcd 制御}
\Rightarrow
\text{素因子支持制御}
\Rightarrow
\text{基底単位互素}
\Rightarrow
\text{局所 comaximal}
}
\]

この一本道じゃ。

わっちの見立てでは、これはかなり筋がよい。
Petal 実装で得た「基底単位を吸収して素性を見る」補題群に、宇宙式側から正当な入口を作れる。
つまり、 **宇宙式は Petal 局所化理論の発生源として読める** 。
